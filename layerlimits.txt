// KEY FINDING
Token volume per layer cannot be generalized. A single model is wrong.
Game engines run 10x the token volume of CLI tools.
ML pipelines peak at Layer 5, not Layer 3 — perf is the real cost.
Embedded systems delay trust crossover to Layer 4 — no runtime flexibility.
APE itself has the heaviest self-repair layer of any archetype — it is the product.
The phase engine is identical for all 10. The weight distribution is unique to each.
Archetype-aware profiling is required to model context demand accurately.
